# 🚀 LangChain Memory Practice

This repository demonstrates different memory techniques in LangChain for building context-aware AI applications.

## 🔍 Memory Types Implemented
- ConversationBufferMemory → Stores full conversation
- ConversationBufferWindowMemory → Stores recent interactions
- ConversationSummaryMemory → Summarizes conversation

## 🛠 Tech Stack
- Python
- LangChain
- OpenAI

```bash
pip install langchain openai
